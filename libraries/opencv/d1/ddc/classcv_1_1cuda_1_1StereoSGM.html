Detailed Description

The class implements the modified H. Hirschmuller algorithm [126]. Limitation and difference are as follows:

  • By default, the algorithm uses only 4 directions which are horizontal and vertical path instead of 8. Set mode=StereoSGM::MODE_HH in createStereoSGM to run the full variant of the algorithm.
  • Mutual Information cost function is not implemented. Instead, Center-Symmetric Census Transform with \(9 \times 7\) window size from [249] is used for robustness.

Parameters

disparityOutput disparity map. It has the same size as the input images. StereoSGM computes 16-bit fixed-point disparity map (where each disparity value has 4 fractional bits).
