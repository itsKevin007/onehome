Detailed Description

Class that manages the extraction and selection of features.

[231] Feature Extraction and Feature Set Refinement (Feature Processing and Feature Selection). See table I and section III C [163] Appearance modelling -> Visual representation (Table II, section 3.1 - 3.2)

TrackerContribFeatureSet is an aggregation of TrackerContribFeature

The modes available now:

  • "HAAR" – Haar Feature-based

The modes that will be available soon:

  • "HOG" – Histogram of Oriented Gradients features
  • "LBP" – Local Binary Pattern features
  • "FEATURE2D" – All types of Feature2D
